# Site Preview de Vídeos

Projeto desenvolvido em HTML puro com sistema de previews de vídeos, navegação entre páginas e links externos para vídeos completos.

## Sobre o projecto

Este site foi criado com fins de estudo e prática de HTML, utilizando:
- links internos
- links externos
- imagens clicáveis
- organização de páginas
- iframes
- tabelas
- vídeos incorporados do YouTube

## Conteúdo utilizado

Os vídeos utilizados pertencem ao Curso em Vídeo, criado por Gustavo Guanabara.

Conteúdo utilizado apenas para fins educativos.

## Tecnologias utilizadas

- HTML5
